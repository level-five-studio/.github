# 👋 Welcome to Level Five Studio

We build modern, production-grade web products using a carefully chosen TypeScript-first stack — fast, maintainable, and automation-heavy by design.

---

## 🛠 What We Build With

| Layer | Technology |
|---|---|
| Language | TypeScript |
| UI | React + Base UI (headless) + SCSS Modules |
| Routing | React Router (SPA / SSG / SSR) |
| Data & State | TanStack Query · Zustand · React Context |
| Build & Test | Vite · Vitest · Playwright |
| Monorepo | Turborepo |
| CMS | Headless, git-backed |
| Packages | GitHub Packages · npm |
| Versioning | Changesets · Semantic Release |

## ⚙️ How We Work

- **Trunk-based development** — small, frequent commits to `main` behind feature flags ([PostHog](https://posthog.com) / [StatSig](https://statsig.com))
- **Automation-first** — CI/CD handles linting, type-checking, testing, and releases via [GitHub Actions](https://docs.github.com/en/actions)
- **AI-driven** — we integrate AI tooling and agents into daily development workflows
- **Strong DX** — first-class tooling: `pnpm`, `oxlint`, `prettier`, `lefthook`, `commitlint`, and more
- **Full test coverage** — Playwright E2E + >75% unit/integration coverage
